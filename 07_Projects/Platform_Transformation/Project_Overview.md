# Platform Transformation — Project Overview

## Background
2026-07-02 起，妹與 ChatGPT（扮演 CEO 角色）討論把「賣書網站」升級為「AI 驅動內容公司」，
產出大量策略對話（角色分工、AIOS 制度、內容資產三層模型、ST 交付模式建議等）。
對話中夾雜大量 CEO 自我反省與重複鋪陳，妹要求 CC 先確認內容真偽、過濾廢話後再整理成正式文件。

## Objective
把 ChatGPT 的策略構想，收斂成「跟現有系統事實一致」的正式決策，寫入 AIOS，
而不是原文照搬一份對話紀錄。

## Scope
- AI 分工制度（CEO / CTO / 其他 AI 角色邊界）
- 內容資產模型（Book / Series / Academy 三層）
- ST / SS / SD 三站定位
- 會員系統定位方向
不含：實際程式碼重構、MongoDB schema migration（待妹對開放問題拍板後才開新任務）。

## Stakeholders
- 妹（董事長，最終拍板者）
- ChatGPT（CEO 角色：策略、商業模式、品牌、內容品質驗收）
- Claude Code（CTO：工程執行、事實查核、文件整理、技術盤點）
