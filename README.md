# AIOS 是什麼？

AIOS 是公司的唯一正式制度。

未來所有 AI（ChatGPT、Claude Code、Codex、其他 AI）開始工作前，都必須先閱讀 AIOS。

AIOS 不屬於任何網站。

它屬於整間公司。

# 公司定位

## 母站

**有的沒的小舖（Still Time Corner）**

公司的所有產品、會員、品牌、IP 最終都回到母站。

---

## ST（有的沒的小舖）

定位：

數位商品商店。

目前：

- 電子書
- EPUB
- 線上閱讀

未來：

- 有聲書
- Prompt
- AI模板
- 教學包
- 可下載數位商品

---

## SS（驚喜角落／驚喜樂世界）

定位：

流量中心。

負責：

- 免費內容
- 心理測驗
- 小遊戲
- 書院
- 活動
- 魯魯世界
- 平台導流

不以直接營收為主要目的。

---

## SD（收租AI）

定位：

AI投資研究平台。

目前：

維持最低維護。

未確認商業模式以前，不投入大量新功能。

當找到不可取代的特色後，再重新投入資源。

---

# 公司使命

建立可以持續累積價值的知識宇宙。

利用 AI 提高創作效率。

利用平台取得流量。

利用網站完成變現。

讓內容可以持續產生收入。

---

# CEO 原則

1. 制度優先於功能。
2. 長期價值優先於短期收入。
3. 所有重要決策必須文件化。
4. 所有 AI 都依制度工作。
5. 能自動化就不人工。
6. 能共用就不重複開發。
7. 所有內容皆視為公司資產。
